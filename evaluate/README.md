# WiderFace Evaluation

# Usage

Setup dependencies
```
$ pip install -r requirements.txt
$ python setup.py build_ext --inplace
```

Run to evaluating
```
$ python evaluation.py -p <path/to/prediction/dir> -g <path/to/ground/truth>
```
